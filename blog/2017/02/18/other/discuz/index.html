window下Discuz论坛环境搭建

2017-02-18
  • 其他
  • 其他
  • 豆

Discuz是一个基于php的论坛的开源框架，本文介绍一下window下Discuz搭建的过程。
首先先到官方论坛下载Discuz包，我这里用的是3.2版本下载地址
然后下载PHPnow，PHPnow包括window下Apache + PHP + MySQL 环境套件包。
解压后放到你喜欢的目录下，直接运行Setup.cmd
选择apache和mysql版本

之后会开始解压文件

解压完成后会
初始化数据库和服务器，设定好服务器密码就OK了

然后将解压出来的Discuz包里面的upload下所有文件复制到htdocs文件夹下
打开浏览器输入http://127.0.0.1/install/，就会开始discuz程序的安装，按说明一步步安装即可。
