window下mysql有两种版本，一种是msi，一种是zip的，msi的安装包只需要一步步按流程走就可以安装好，这里说一下zip类型的安装方式。

先配置mysql目录下的
my-default.ini，或者自己建立一个my.ini文件，
在其中修改或添加配置

进入bin下执行mysqld -install (注意用管理员的身份执行)
安装成功会有Service successfully installed提示

然后初始化data数据
还是在bin下执行mysqld --initialize-insecure自动生成无密码的root用户
(mysqld –initialize自动生成带随机密码的root用户)。
注意:data文件夹不为空是不能执行这个命令的。

初始化完数据，就可以启动服务
在bin下执行net start mysql
