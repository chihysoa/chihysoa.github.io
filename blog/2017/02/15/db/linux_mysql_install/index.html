linux安装mysql比较window下稍微要麻烦一点。

第一种方式

Redhat/Fedora/CentOS下可以用yum install mysql命令安装，有时候坑需要额外安装yum install mysql-server

ubuntu下用sudo apt-get install mysql-server命令，中途会要你设置root用户的密码

第二种方式比较麻烦
