Error:Cause: failed to find target with hash string ‘android-23’ in:

android studio未找到android版本包，打开SDK Manager发现已经安装了，搞了好久没找到原因，偶然的打开android studio的Android SDK看到却发现没有安装，于是使用android studio安装一下，搞定。
还有可能是 Android\sdk\temp 目录是只读的，下载程序无法写入，去掉只读属性就行。

关于版本号

在使用了gradle以后，应用的版本号就不是写在AndroidManifest.xml里面了，而是写在应用下的build.gradle里面
