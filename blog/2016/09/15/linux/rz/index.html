window下经常会要上传文件或者下载文件到linux系统上，用的比较多的是rz和sz，说下常用的安装和使用。

安装

CentOS 7、RHEL 7、Fedora：yum -y install lrzsz
Ubuntu下可以用：apt-get install lrzsz

使用

文件会自动下载到本地，默认为系统的下载目录，也可以通过配置修改默认下载目录

上传

输入

会自动打开一个文件窗口，选择需要上传的文件即可（可多选）
