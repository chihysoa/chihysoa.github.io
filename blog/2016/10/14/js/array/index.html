javascript的数组是我们经常会使用的一个对象。这里对数组的一些常用方法做一个简单的介绍。

创建

javascript中所有的数组都是Array对象的实例。
创建一个数组，我们可以通过他的构造函数，也可以使用直接量。

在es6中还有个Array.of方法可以用来创建数组

静态方法

Array.from(es6)

用来将一个类数组对象转成数组对象。

Array.from(arrayLike[, mapFn[, thisArg]])

Array.isArray

判断一个对象是否是数组

Array.of(es6)

用于创建数组

实例方法

concat

var new_array = old_array.concat(value1[, value2[, …[, valueN]]])

将传入的数组组装成一个新的数组并返回，不会影响原数组

copyWithin

arr.copyWithin(target[, start[, end]])

在数组内复制从start(包含start)位置到end(不包含end)位置的一组元素覆盖到以target为开始位置的地方，改变的是数组本身，最后一个参数为可选参数，省略则为数组长度。

entries(es6)

arr.entries()

返回一个遍历器对象，数组的键值对。

every

arr.every(callback[, thisArg])

用来判断数组元素是不是都符合某个条件

fill(es6)

arr.fill(value[, start = 0[, end = this.length]])

对目标数组用指定的元素填充，改变的是数组本身

filter

var new_array = arr.filter(callback[, thisArg])

过滤器，用来筛选符合条件的元素，不影响原数组

find(es6)

arr.find(callback[, thisArg])

返回第一个符合条件的元素

findIndex(es6)

arr.findIndex(callback[, thisArg])

返回第一个符合条件的下标

foreach

arr.forEach(callback[, thisArg])

遍历目标数组

includes(es6)

var boolean = array.includes(searchElement[, fromIndex])

判断目标数据是否包含某个元素

indexOf

arr.indexOf(searchElement[, fromIndex = 0])

查找元素在目标数组的位置，没有返回-1

join

str = arr.join([separator = ‘,’])

根据所传入的分隔符(默认为,)，将数组转为字符串

keys(es6)

arr.keys()

返回所有的key值组成的遍历器对象

lastIndexOf

arr.lastIndexOf(searchElement[, fromIndex = arr.length - 1])

返回数组中最后一个符合指定元素的下标

map

arr.map(callback[, thisArg])

遍历数组，并根据所传入的回调函数，返回新的数组，不影响原数组。

pop

arr.pop()

将目标数组的最后一个元素从数组中移除，返回该元素，改变的是数组本身

push

arr.push(element1, …, elementN)

往目标数组尾部加入指定的元素，改变的是数组本身

reduce

arr.reduce(callback[, initialValue])

遍历数组，将回调函数的返回值为累积结果，并且此返回值在下一次调用该回调函数时作为参数提供。initialValue作为第一次调用回调函数的参数。

reduceRight

arr.reduceRight(callback[, initialValue])

与reduce相同，遍历方式倒序

shift

arr.shift()

将目标数组的第一个元素从数组中移除，返回该元素，改变的是数组本身

slice

arr.slice([begin[, end]])

从数组中以新数组的方式返回选定的元素，不影响原数组。

some

arr.some(callback[, thisArg])

判断数组中是否存在符合条件的元素。

sort

arr.sort([compareFunction])

对数组进行排序(默认为自然顺序)，不影响原数组。

splice

arrayObject.splice(index,howmany,item1,…..,itemX)

从数组中添加/删除项目，然后返回被删除的项目，index必需。整数，规定添加/删除项目的位置，使用负数可从数组结尾处规定位置。howmany必需。要删除的项目数量。如果设置为 0，则不会删除项目。item1, ..., itemX可选。向数组添加的新项目，改变的是数组本身。

unshift

往目标数组头部加入指定的元素，改变的是数组本身

values(es6)

arr.values()

返回一个遍历器对象，数组的所有元素。

实例属性

length

返回数组的长度
