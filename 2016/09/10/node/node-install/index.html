linux下安装nodejs

  • node
node

作为一个程序员，linux系统必须要会折腾，作为一前端nodejs'必须要研究，给大家科普下linux下安装nodejs，linux下安装nodejs`有三种方式，下面分别对这三种方式一一介绍

首先到官网下载–>

第一种（直接法）

个人这种方式比较简单

下载编译好的二件制文件包Linux Binaries (.tar.xz)
下载完成后解压文件到某个你喜欢的目录node-x.x
将node-x.x/bin路径加到系统PATH里面
开始使用
恩,就是这么简单。

第二种（源码编译）

这种方式需要python支持

*这次下载的文件包不一样，下载的是源码文件包`Source Code`*
*下载完成后解压文件到某个你喜欢的目录`node-x.x`*
*`cd node-x.x`进入nodejs的源码目录*
*'./configure'*
*'sudo make && make install`*
*等待编译完毕就ok了'*

注：如果都执行完毕后node命令无效，可能是node未加入到环境变量，可以尝试执行cp /usr/local/bin/node /usr/sbin/
