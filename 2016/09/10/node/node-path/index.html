linux设置环境变量

  • linux
linux

在平时使用的时候，经常会遇到需要设置环境变量，linux下设置环境变量有两种方式

全局

全局的对所有用户都可以的使用的PATH: 可以通过修改配置文件： /etc/bashrc 和 /etc/profile 来时配置，全局的PATH。
例如： vi /etc/profile
在最后后加一行：PATH=THE_SET_GLOBLE_ENV_PATH:$PATH;
接着换行： export PATH
如果使其立即生效： source /etc/profile

对用户

对于普通用户也可以设置，自己的路径：在自己的home目录中：ls -a | grep bash
会找到两个文件： .bashrc 和 .bash_profile 。
同样可以像修改 /etc/profile一样，修改 .bash_profile 来配置你的个性PATH 环境。
同样修改后，若不重启，就使其立即生效的话，不要忘了 source .bash_profile。
