es6入门--基本扩展(一)

  • es6
javascript

es6又名es2015规范已经出来已经有一段时间了，我也使用了一段时间了，在这里介绍一些es6的新特性和亮点，有些是es7草案的语法，关于es7因为还处于草案阶段，使用的取舍看个人了。

const/let

const/let是es6新引入的两个关键字，他们用来声明变量，用了替代以前的var，对没错就是替代。
那么它们是什么呢。
const 用来声明的变量表示这是一个常量，一旦声明后就不能再被赋值，因此必须在声明时对其进行赋值。

let用来声明局部变量，es6中引入了块级作用域，即一个{}就代表一个块，在这个块里用let声明的变量只在这个{}所在的块中有效。

还有const和let不存在变量的提升，这一点需要特别注意。

解构赋值

解构赋值就是等号两边相同结构的变量赋值，解构包括数组解构和对象解构。
let [a，b]=[1，2];
解构结果a=1，b=2。这个就是数组解构，数组解构就是按数组下标去依次对应的赋值
let {a:b} = {a:1};
解构结果 b=1对象解构的时候会按相对应的key来解构。

箭头函数

箭头函数是对function的简写
(a，b)=>{let c=a+b;console.log(c)}
这是一个简单的箭头函数，它等价于
function(a，b){let c=a+b;console.log(c)}
相比较原来的传统写法，他会继承父作用域的this和arguments，这里解决了我们经常会要使用es5里的.bind(this)。

数组扩展

Array.of

Array.of是对new Array()的扩展。
new Array()当传入一个参数时 new Array(5)会生成一个长度为5的空数组，但是我们想要的是一个元素为5，长度为1的数组，怎么办呢？直接使用字面量[5]。为此es6中引入了Array.of方法用于创建数组

Array.from

Array.from作用将一个类数组对象转换成一个数组对象和扩展运算符效果一样。
Array.from(arguments)
和扩展运算符不同的是，它可以将两种对象转成数组，一种是和扩展运算符一样，部署了iterator的对象。还有一种是类数组对象，js里面有很多类数组对象，比如arguments，nodelist等都是类数组对象，它们有类似数组一样的key值（下标）
也有length，但是它们不具备数据的一些方法，如shift，slice等，es6之前，我们可以通过 [].shift.slice.call(arguments)来把一个类数组对象转换成数组对象，现在使用Array.from(arguments)就可以得到一个数组对象。

Array.find/findIndex

find/findIndex用来查找第一个符合条件的元素（下标），未找到返回undefined(-1)，它的参数是一个回调函数

Array.fill

Array.fill用于填充数组，已有的元素会被覆盖

它还接受第二个第三个参数
new Array(3).fill(5,1,2);//[,5,]
第二个参数为起始下标，第三个参数是终止下标

entries/values/keys

es6数组实例还提供了entries/values/keys，三个遍历器，可以用for..of..来循环
